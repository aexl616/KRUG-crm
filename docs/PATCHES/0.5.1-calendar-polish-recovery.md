# Patch 0.5.1 — Calendar Polish Recovery

**Тип:** Fix + Recovery

## Причина проверки

После изменения порядка коммитов требовалось убедиться, что функции Calendar Polish не были затёрты более ранней версией Studio Calendar Core.

## Проверенные версии

- `1488853` — Patch 0.5, Calendar Polish;
- `bcd5502` — Patch 0.6, UI/UX;
- `a71cf51` — текущий `main` на момент recovery-проверки.

## Результат

Текущий `main` содержит функции и стили из Patch 0.5 и безопасные визуальные улучшения Patch 0.6. Повторное наложение старого кода не потребовалось.

Подтверждено наличие:

- режимов календаря «Неделя» и «День»;
- фильтров по сотруднику, статусу и услуге;
- проверки конфликтов записей;
- информативных карточек календаря;
- контекстных действий в правой панели;
- сохранения календарных настроек;
- защиты от дублей связанных платежей;
- безопасного удаления записи без удаления платежа;
- объединения клиентов из `clients`, `bookings` и `payments`.

## Сохранено из Patch 0.6

- экран «Сегодня»;
- ближайшая запись и расписание дня;
- свободные окна;
- локальные уведомления;
- глобальный поиск;
- command palette;
- совместимые UI-стили.

## Не переносилось

Старые версии `app.js` и `styles.css` не накладывались повторно, поскольку текущая реализация уже содержит необходимые изменения. Это предотвращает дублирование логики и возможные регрессии.

Не добавлялись роли, комнаты, Telegram, онлайн-запись и drag & drop.
